# TypeCraft - Typing Trainer

A simple, offline typing trainer inspired by MonkeyType. No account needed, no internet required after download.
It helps improve your typing speed

## How to use

1. Download `typecraft.html`
2. Open it in any web browser (Chrome, Firefox, Edge)
3. Click the typing area or press any key to start
4. Type the words you see, green means correct, red means wrong
5. Press `Tab` to restart at any time

## Features

- **WPM tracking** - live words-per-minute shown as you type
- **Accuracy & errors** - tracked in real time
- **Personal best**  - saved automatically, flagged when you beat it
- **30-word checkpoint** - a toast pops up mid-test showing your speed at 30 words
- **Session history** - every completed test is saved locally in your browser
- **WPM trend chart** - shows your progress over time in the history panel
- **Three time modes** - 30s, 60s, or 120s

## Controls

| Key | Action |
|-----|--------|
| Any letter | Type |
| Space | Move to next word |
| Backspace | Delete last character |
| Tab | Restart test |

## Notes

- All data is saved locally in your browser (`localStorage`)  - nothing is sent anywhere
- Clearing your browser data will erase your history
- Works fully offline once downloaded
